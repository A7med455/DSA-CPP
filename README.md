# DSA-CPP

## 📘 Description

This repository contains implementations of core Data Structures and Algorithms (DSA) concepts in C++.

It focuses on fundamental structures such as arrays, linked lists, stacks, queues, and binary trees, with both iterative and recursive implementations to reinforce understanding and problem-solving skills.

## 🧩 Topics Covered

* Arrays
* Binary Trees
  * Iterative Implementation
  * Recursive Implementation
* Linked Lists
  * Singly Linked List
  * Doubly Linked List
* Queues
  * Queue
  * Circular Queue
  * Linked List Based Queue
* Stacks
  * Stack
  * Linked List Based Stack

## ⚙️ Language

C++

## 🚀 Purpose

To build a strong foundation in DSA through hands-on coding and continuous practice.
